石虎

淺山森林之王

進入了「石虎森林」當然要認識這森林之王—石虎。石虎生活在淺山森林，與生活在深山的「台灣雲豹」， 同樣都是「貓科動物」，但是雲豹已經在台灣的山林間消失很久、不見蹤跡，如果「台灣雲豹」真的絕跡了， 石虎將會孤單的成為台灣唯一的貓科動物，不再加緊石虎的保育工作，台灣的山林間就再也沒有山貓了。 關心石虎、保育石虎，就從認識牠開始。

跟我一起!保護石虎!!
